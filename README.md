## MCA DBMS Mini Project

This project demonstrates:

- Row Level Locking
- Deadlock Handling
- Optimistic Locking
- Rollback Handling
- Isolation Levels
- Auto Seat Release

## Technologies
- MySQL
- SQL Transactions
- Concurrency Control

## Features
- No double booking
- Safe concurrent booking
- Parallel seat handling
- Waiting queue support
